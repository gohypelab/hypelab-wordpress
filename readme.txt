== Description ==
HypeLab for WordPress is a plugin that allows you to add HypeLab banner, native, and rewarded placements to your WordPress site.

== Installation ==
1. Download the plugin zip file from our [GitHub repository](https://github.com/gohypelab/hypelab-wordpress)
2. Upload the plugin to your WordPress installation: Plugins > Add New Plugin > Upload Plugin
3. Activate the HypeLab for Wordpress plugin

== Usage ==
First, configure the HypeLab plugin by visiting to HypeLab > Settings in your WordPress dashboard.
Then, enter your HypeLab property slug and select the appropriate URL and environment. 
After saving your changes you are now ready to add placements to your site!
Visit our [documentation site](https://docs.hypelab.com) to view documentation about adding banner, native, and rewarded placements.
